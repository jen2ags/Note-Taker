# Note Taker 

## Purpose
The purpose of creating the Note Taker is to be able to create and save notes into our note taker and then be able to edit the current notes that we have or when they are completed, delete them. 

## Built With
* HTML
* CSS
* JavaScript
* Express.js
* UUID
* heroku
